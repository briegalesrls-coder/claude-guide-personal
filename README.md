# Claude Code Guide — Personale

Guida didattica personale Claude Code v2.0 + AI Engineering avanzato.

## Come leggere

- **Desktop**: apri `index.html` con doppio click
- **Mobile/Tablet**: clona il repo, oppure visualizza file via GitHub mobile app

## Contenuto

29 sezioni navigabili:
- Fondamenti (regole d'oro, token economics, gerarchia cervello)
- I 9 pezzi del cervello Claude (CLAUDE.md, agent, skill, MCP, hook, style, command, memory)
- Decision making (skill vs agent vs command, multi-progetto, prompt patterns)
- Anti-pattern (10 errori comuni)
- Per il futuro (setup nuovo progetto, versioning, roadmap esperto AI)
- AI Engineering avanzato (Claude API, architetture agentiche, eval, sicurezza, RAG, caso studio Copilot Direttore)
- Reference (quick card, glossario)

Tutti i capitoli AI Engineering hanno **esempi accademici completi** applicati al Copilot Direttore reale (codice Python, SQL, JSON).
